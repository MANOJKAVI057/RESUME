Projects

WORD PUZZLE

Description: The Word Puzzle Webpage is an interactive web-based application designed to provide users with an engaging word puzzle experience. Developed using HTML, CSS, and JavaScript, the game features a dynamic and responsive interface where users can search for hidden words in a grid that changes with each session. The frontend ensures smooth play with real-time score tracking, visual feedback, and intuitive design to enhance user engagement. To control access and manage user sessions, a secure login and registration system is implemented using PHP and MySQL. User credentials are stored securely in the MySQL database, and session management is handled through PHP to ensure authenticated access. Additionally, an optional admin panel is available for maintenance purposes, allowing administrators to manage user accounts and view score data stored in the database. This project showcases a complete full-stack solution integrating frontend interactivity with backend data handling for a seamless and secure user experience.

SMART RETAIL

Description: The Smart Retail Application is an intelligent system built using Python that streamlines the analysis and visualization of retail data such as cars, bikes, or other commercial products. This application includes a secure login system to authenticate users and restrict access, with credentials stored and managed using a MySQL database. Once logged in, users can access various modules including a Data Upload Page for importing retail datasets, a Data Preprocessing and Extraction Page that cleans and prepares the data for analysis, and a Visualization Page where analytical results are displayed in the form of insightful graphs. These graphs are generated based on machine learning or statistical algorithms implemented in Python, offering valuable insights such as sales trends, category comparisons, or customer behavior. The backend handles data storage and processing, while the frontend provides a user-friendly interface for seamless navigation across modules. This project demonstrates the integration of data-driven decision-making within a retail environment using full-stack Python development with MySQL support for data maintenance.
